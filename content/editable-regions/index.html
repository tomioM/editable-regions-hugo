Edit what you see with Editable Regions

Simple HTML content is editable in real time, right on the page. Content editors will see these regions marked by yellow borders here in the Visual Editor. You can configure exactly what is editable with this class in your code.

Inline editables

p, h1 - h5, a, span, button, li, img

Adding class="editable" to any inline HTML element will make it editable in the Visual Editor. You’ll see a yellow outline indicating editable elements. Try it for yourself in the window below!

For text, you’ll see simple formatting options. Links and buttons receive a drop-down menu to manage the link and descriptions.

The images drop-down helps you add or change images for those already in your CloudCannon folders. It also makes it easy to update image metadata details such as alt-text, description and title.

Block editables

div, section, aside, main

Editable Regions are flexible and can work across larger HTML elements, too. Adding the "editable" class to a div or section makes the tags within it completely editable.

The block toolbar gives a wide range of editing options — you can change heading tags, start a quote block, create a list, and more.

For complex structures or page layouts, we recommend you use multiple editable regions across your individual elements.
